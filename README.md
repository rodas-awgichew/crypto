# Crypto

Crypto is a modern web application project aimed at revolutionizing digital finance through secure, transparent, and decentralized blockchain solutions. The repository demonstrates how advanced web technologies, animation, and data-driven design can deliver real-time crypto insights and user engagement.

## Tech Stack

- **HTML5**: Semantic markup for clear structure and SEO
- **CSS3**: Custom styles with the Poppins font and radial gradients; responsive layouts and slick visual effects
- **Bootstrap 5**: Grid, utilities, and ready-to-use components for rapid development
- **JavaScript/GSAP**: GSAP powers dynamic UI effects, hero transitions, scroll-based animations, and floating elements.
- **AOS (Animate On Scroll)**: Smooth element entrance animations for interactive experiences


### Modern Web Experiences
Animated hero sections, floating graphics, image carousels, and glowing text accents for visual appeal and engagement.

### Interactive Blog & Resources
Features curated insights, articles, and perspectives on blockchain, AI, and digital innovation.
